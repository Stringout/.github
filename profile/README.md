# Stringout

Stringout is media organization software for post-production teams. It helps
teams assemble footage faster by indexing media and making every drive
searchable without uploading originals. Pro video editors use it to create
editor-ready handoffs to get to the timeline faster.

## What Stringout does

- Indexes media from connected drives and folders in the background.
- Uses on-device AI to transcribe, tag, and organize footage metadata.
- Builds a single searchable catalog across current and previously connected
  drives.
- Connects indexed metadata to your team workspace for fast discovery.

## Why teams use it

- Get to the timeline faster with one search bar for your media.
- Keep originals on your own storage with a no-upload workflow.
- Run media prep locally with Apple Silicon acceleration.
- Avoid cloud processing overhead for core indexing workflows.

## How it works

- **Stringout Agent (macOS):** Runs in your menu bar and indexes local media.
  Orchestrates probing, transcription, vision analysis, and embeddings.
- **Stringout web app:** Lets teams browse and search indexed footage from
  anywhere.
